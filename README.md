# A Collection of vim Configurations

Current modifications:

* map `jk` to `<ESC>`
* map mapleader to `" "`
* spellcheck
* onedark theme with polyglot
* lightline and air-line
* tab preferences
* relative line numbering
* fzf support
* gitgutter support
* NERDTree

## Setup
 
* Clone the repo as `.vim` in your home directory. 
* Run `./setup.sh`
